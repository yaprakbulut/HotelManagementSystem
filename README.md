# Hotel Management System

## How to run?

1. Download the the zip file
2. Extract the file and copy hotel folder
3. Paste inside root directory(for xampp xamp/htdocs)
4. Create a database with name hotel
5. Import hotel.sql file(given inside the zip package in sql file folder)
6. Run the script http://localhost/hotel/login
7. Username: admin Password: 1234

You can use the username and passwords in the users table to login with another user.
